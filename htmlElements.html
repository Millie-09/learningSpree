HTML Elements

So far, I have come across some really cool HTML elements.
I have had the opportunity to play around with a few, and come across some more in the course of my assignments/challenges with eHAcademy. I have learnt that there is no use trying to cram all the elements, as there are actually alot and functions too. Instead, with the few that I have come across, I take note of these key facts:

  • Is this element self-closing or not.
  • Is this element an in-line or block-level element

Once I have figure this two out, I make use of Google for the rest. Let's say I want to make my text italics, I could google it and get elements that can make this happen for me.

Example: To do this assignment, I noticed that I couldn't code the tags and still have them displayed on my webpage, so, I Google-d "how do I write a tag in a list and have it displayed as text". This supplied me with different url to pages where I got the information I needed. Google is the GOAT!
